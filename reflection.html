Reflection on This Project

The purpose of this project was to learn about how briges, specifically in the Philadelphia area can build and disrupt communities. I decided to make a website because I thought that could be an effective way of presenting some research about these questions.

Technical Aspect

Doing this project not only taught me about bridges but also more about web development. This website was made using HTML CSS and JavaScript. For the map I used a opensource JavaScript called Leaflet.js. Leaflet makes it simple to make maps and add pop up information to specific points on those maps. You can also set a coordinate for it to in essence jump to every time the website is refreshed. I did not use any css libraries such as Bootstrap because I wanted more control of the customization of it. The navbar and other aspects of the design of this website was inspiration taken from the robotics team 23396 Hivemind's website however an older version not how it is right now. This website is hosted by github pages because that was a very simple (and free) option. To see my code, you can look at this websites repository.

Non-technical Aspect

I did like making the website more then actually doing the research or doing the general point of this project however that was also fun I suppose. I learned a lot about bridges. And specifically bridges in the Philadelphia area. I learned how they can build communities by connecting people and hurt communities when they collapse.
